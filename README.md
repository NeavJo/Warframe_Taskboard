# Warframe Taskboard

> 以**任务看板**与**攻略收集**为核心的 Warframe 玩家多合一工具包。

还在为每天忘记做日常、每周漏掉周常、查攻略要在多个网站间反复切换而烦恼？Warframe Taskboard 把这些需求整合到一个轻量级 Web 应用中，以奥罗金暗色科幻美学呈现，让你既高效又沉浸地管理你的 Tenno 日常。

---

## ✨ 核心功能

### 📋 任务看板
- **每日日常** 与 **每周周常** 双面板并排展示
- 任务**拖拽排序**（左侧手柄拖动），管理模式下可编辑/删除
- 完成/未完成状态自动统计，进度条可视化
- **自动重置**：每日 0 点重置日常、每周指定工作日重置周常（仅未完成项重置）
- 自定义任务图标、强调色（金/蓝/黄三色系）

### ⏰ 定时提醒
- 添加带日期时间的提醒项，支持"X 小时后"快捷设置
- 到达预定时间后卡片**自动激活**并发出脉冲发光动画
- 点击激活的卡片即可标记完成
- 提醒超时 30 分钟未处理则**自动清理**

### 🌐 内置浏览器
- 默认打开**灰机 Wiki**，方便查阅中文攻略
- 其他站点（如官方 Wiki、市场）以**新标签页**方式跳转，不打断当前工作流
- 内置 URL 输入栏、刷新、首页按钮

### ⚙️ 设置与同步
- **JSON 导入/导出** 任务数据，方便跨平台/跨设备同步
- 包含看板任务与提醒数据完整导出
- 纯本地存储（localStorage），无需账号、无需联网

---

## 🎨 设计亮点

- **奥罗金美学**：深灰/黑底色 + 金色/蓝色/电磁黄色三强调色，呼应 Warframe 视觉语言
- **发光交互**：导航触发按钮带脉冲发光，激活提醒卡片带呼吸动画，完成徽章带金色光晕
- **响应式自适应**：
  - 桌面端（≥900px）：顶栏 + 左侧栏 + 双栏看板
  - 移动端（<900px）：抽屉式侧边栏 + 单栏 Tab 切换
  - 手机竖屏（<480px）：紧凑间距适配小屏
- **跨滚动条美化**：自研 WebKit/Firefox 双引擎滚动条样式，与主题融为一体

---

## 🛠 技术栈

- **HTML5 / CSS3 / 原生 JavaScript（ES6+）**
- **零构建工具**：无 Webpack/Vite，开箱即用
- **零运行时依赖**：仅引入 Google Material Icons 字体
- **本地存储**：基于 `localStorage` 持久化所有数据
- **SPA 架构**：单页应用，多视图切换不刷新

## 🚀 快速开始

### 本地运行
1. 克隆或下载本项目
2. 进入 `wf-taskboard-web/` 目录
3. 用任意 HTTP 服务器打开 `index.html` 即可

```bash
# 示例：Python 启动本地服务器
python -m http.server 8080
# 浏览器访问 http://localhost:8080/
```

> 💡 也可直接双击 `index.html`，但部分浏览器功能（如 iframe 嵌入）可能因 `file://` 协议受限，推荐使用本地服务器。

### 使用说明
1. 打开应用后默认进入"看板"
2. 点击右上角"管理"按钮可新增、编辑、拖拽排序任务
3. 切换到"提醒"页添加定时提醒
4. 切换到"浏览器"查阅攻略
5. 切换到"设置"导出/导入数据进行跨设备同步

## 📌 项目背景

本项目最初为 Flutter 应用（Windows .exe + Android .apk 双端），后迁移至 Web 平台以扩大可访问性。原始 Flutter 版的核心理念——**任务看板 + 攻略收集 + 暗色科幻美学**——在 Web 版中得到完整保留与增强。
